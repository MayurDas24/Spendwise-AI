# 🚀 Spendwise-AI

<p align="center">
  <b>AI-Powered Expense Tracker built using the PERN Stack</b>
</p>

<p align="center">
  Smart budgeting • AI insights • Analytics • Financial health tracking
</p>

---

## ✨ Features

* 🔐 JWT Authentication & Secure Login
* 💸 Income & Expense Management
* 📊 Interactive Analytics Dashboard
* 🧠 Gemini AI Financial Insights
* 📈 AI Spending Analyzer
* 🎯 Monthly & Weekly Budget Tracking
* 🧾 Smart Transaction Filtering & Search
* 🏷️ Custom Categories with Icons & Colors
* 📉 Budget Verdicts & Savings Recommendations
* 📂 Persistent AI History using PostgreSQL JSONB
* 📱 Fully Responsive Modern UI

# 🚀 Installation

## Clone Repository

```bash
git clone https://github.com/MayurDas24/Spendwise-AI.git
```

## Backend Setup

```bash
cd backend
npm install
npm run dev
```

## Frontend Setup

```bash
cd frontend
npm install
npm run dev
```
